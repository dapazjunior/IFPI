API Cidades — Frontend

GET

/cidades

Lista todas as cidades. Filtre por UF (opcional).

GET

/cidades/{id}

Retorna uma cidade pelo ID.

POST

/cidades

Cria uma nova cidade.

PUT

/cidades/{id}

Atualiza uma cidade existente pelo ID.

DELETE

/cidades/{id}

Remove uma cidade pelo ID.

GET

/estados/{sigla}/cidades

Lista cidades de um estado pela sigla.
